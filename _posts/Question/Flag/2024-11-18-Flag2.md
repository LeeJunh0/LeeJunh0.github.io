# 풀이

```c++
int main()
{
    char arr[] = { 'S', 't', 'r', 'u', 'c', 't', 'P', 'o', 'i', 'n', 't' };
    char input;
    std::cin >> input;

    bool flag = false;

    for(int i = 0; i < sizeof(arr) / sizeof(char); i++)
    {
        if(arr[i] == input)
        {
            flag = true;
            break;
        }
    }

    if(flag == true)
        std::cout << "발견";
    else
        std::cout << "미발견";

    // 입력 : P

    // 결과
    // 발견

    return 0;
}
```
---------------------------------------

# 후기

flag방식과 counting방식을 구분해서 써야 한다 배웠다.
