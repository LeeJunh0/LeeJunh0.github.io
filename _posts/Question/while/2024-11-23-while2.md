# 풀이

```c++
int main()
{
    int num;
    std::cin >> num;

    int i = 0;
    while(i++ < 3)
    {
        int j = 0;
        while(j++ < 5)
            std::cout << num;

        std::cout << std::endl;
    }

    // 입력 7
    
    // 결과
    // 77777
    // 77777
    // 77777

    return 0;
}
```
