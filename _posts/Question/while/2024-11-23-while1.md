# 풀이

```c++
int main()
{
    int a, b;
    std::cin >> a >> b;

    while (a <= b)
    {
        std::cout << a << " ";
        a++;
    }

    // 입력 : 3 6

    // 결과
    // 3 4 5 6

    return 0;
}
```
---------------------------------------

# 후기
