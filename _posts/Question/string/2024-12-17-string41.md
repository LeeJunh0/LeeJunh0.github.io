# 풀이

```c++
std::string passArr[5] =
{
    "Jason",
    "Dr.tom",
    "EXEXI",
    "GK12P",
    "POW"
};

int main()
{
    std::string input;
    std::cin >> input;
    
    bool flag = false;
    for (int i = 0; i < 5; i++)
    {
        if (input == passArr[i])
        {
            flag = true;
            break;
        }
    }
    
    if (flag == true)
    	std::cout << "암호해제";
    else
    	std::cout << "암호틀림";
    
    // 입력
    // Jason

    // 결과
    // 암호해제

    return 0;
}
```
---------------------------------------

# 후기

연산자 오버로딩이 되어있어 string간에 == 연산이

strcmp처럼 가능하게 되어있다.
