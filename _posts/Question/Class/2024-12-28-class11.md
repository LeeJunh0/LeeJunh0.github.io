# 풀이

```c++
#include <iostream>

class Array
{
private:
    int count;
    int* arr;

public:
    Array(int _count)
    {
        count = _count;
        
        arr = new int[count];
        for (int i = 0; i < count; i++)       
            arr[i] = rand() % 10;      
    }
    
    void PrintArray()
    {
        for (int i = 0; i < count; i++)
            std::cout << arr[i];
    }
    
    ~Array()
    {
        delete[] arr;
    }
};


int main()
{
    srand(time(NULL));
    
    Array arr1(3);
    Array arr2(5);
    
    arr1.PrintArray();
    std::cout << std::endl;
    arr2.PrintArray();
    
    return 0;
}

```
---------------------------------------

# 후기

라이브러리를 사용하여 하는 문제였어서 srand와 time을 알아보고

rand함수가 먼지 알아보고 풀어본 문제였다.
