# 풀이

```c++
void Recursion(int n);

int main()
{
	Recursion(0);

	return 0;
}

void Recursion(int n)
{
	if (n >= 10)
		return;

	Recursion(n + 1);
	Recursion(n + 2);
	Recursion(n + 3);

	std::cout << "현재 : " << n << std::endl;
}
```
---------------------------------------

# 후기

재귀함수에 너비에 대해 배웠다. 너비란 함수에서 다른 재귀로 갈라지는걸 말하는거였다.
