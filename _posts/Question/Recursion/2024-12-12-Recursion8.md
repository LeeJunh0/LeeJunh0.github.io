# 풀이

```c++
void Recursion(int num);

int main()
{
    int input;
    std::cin >> input;

    Recursion(input);

    // 입력
    // 17
    
    // 결과
    // 1 2 4 8 17

    return 0;
}

void Recursion(int num)
{
    if(num == 0)
        return;

    Recursion(num / 2);
    std::cout << num << " ";
}
```
