# 문제   
---------------------------------------

![Desktop View](/assets/img/Recursion12.png){: w="700" h="400" }

---------------------------------------

# 풀이

```c++
void Recursion(int n);

int level;

int main()
{
    std::cin >> level;
    Recursion(0);

    // 입력
    // 3

    // 결과
    // 012332331233233

    return 0;
}

void Recursion(int n)
{
    std::cout << n;
    if(n == level)
        return;

    for(int i = 0; i < level; i++)
        Recursion(n + 1);
}
```
---------------------------------------

# 후기

재귀를 for문으로 돌리고 그걸 트레이싱 하는 연습을 한 문제였다.
