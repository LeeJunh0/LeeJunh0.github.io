# 풀이

```c++
void Recursion(int i);

int arr[6];

int main()
{
    std::cin >> arr;
    Recursion(0);

    // 결과
    // abcde
    // edcba

    return 0;
}

void Recursion(int i)
{
    std::cout << arr[i];
    if(arr[i] == '\0')
    {
        std::cout << std::endl;
        return;
    }

    Recursion(i + 1);
    std::cout << arr[i];
}
```
---------------------------------------
