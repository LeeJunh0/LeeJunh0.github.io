# 풀이

```c++
int main()
{
    int num;
    std::cin >> num;
    
    int a, b;
    
    a = num % 10000 / 1000 * 10;
    b = num % 100 / 10 + 5;
    
    std::cout << a + b;
    
     // 입력
     // 354129
    
     // 결과
     // 47
    
    return 0;
}
```
---------------------------------------

# 후기

나머지로 해당 자리수 뒤의 숫자들만 남기고 1000자리면 1000을 나눠 해당 자리 값만

구할수 있다 그 방식으로 앞뒤 숫자를 만들고 앞자리는 10자리니까 10을 곱해주었다.
