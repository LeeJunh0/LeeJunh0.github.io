수학문제를 코드로 구현하는 문제다.
이건 나눗셈과 나머지를 활용하는 문제다.

# 문제   
---------------------------------------

![Desktop View](/assets/img/Math1.png){: w="700" h="400" }

---------------------------------------

# 풀이

```c++
int main()
{
    int input;
    std::cin >> input;

    int a, b, c, d;

    // 나눗셈을 하면 앞 자리가!
    // 나머지를 하면 뒷 자리가!

    a = input / 1000;
    b = (input % 1000) / 100;
    c = (input % 100) / 10;
    d = input % 10;

    // 입력
    // 7942

    // 결과
    // 숫자7
    // 숫자9
    // 숫자4
    // 숫자2

    return 0;
}
```
