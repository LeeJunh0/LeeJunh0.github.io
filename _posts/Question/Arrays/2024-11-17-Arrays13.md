# 풀이

```c++
int InputMethod();
void Calc(int a, int b, int c);

int main()
{
    int a, b, c;
    a = InputMethod();
    b = InputMethod();
    c = InputMethod();
    Calc(a, b, c);
    return 0;
}

int InputMethod()
{
    int input;
    std::cin >> input;
    return input;
}

void Calc(int a, int b, int c) { std::cout << a + b + c; }
```
