![Desktop View](/assets/img/Array64.png){: w="700" h="400" }

---------------------------------------

# 풀이

```c++
int main()
{
    int vect[3][4] =
    {
        3,4,1,5,
        3,4,1,3,
        5,2,3,6
    };
    
    int input;
    std::cin >> input;
    int sum = 0;
    for (int i = 0; i < 3; i++)
        sum += vect[i][input];
    
    std::cout << sum;

    // 입력
    // 2

    // 결과
    // 5

    return 0;
}
```
---------------------------------------

# 후기

다차원배열에서 input에 해당하는 vect[input] 값을 전부 합해 출력하는문제 였다.
