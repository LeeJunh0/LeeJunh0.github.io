---------------------------------------

# 풀이

```c++
char Scoring();

int main()
{
    std::cout << Scoring();
    return 0;
}

char Scoring() 
{
    int input;
    std::cin >> input;

    if(90 >= input)
        return 'A';
    else if(80 >= input)
        return 'B';
    else if(70 >= input)
        return 'C';
    else
        return 'D';
}
```
---------------------------------------

# 후기

조건마다 리턴으로 다른 조건에 만족을 해도 데이터를 반복하여 넣지 않도록 유도한거같다.
