배열에 익숙해지기 위한 연습문제19 이다!

서로 다른 배열들의 같은 인덱스 끼리 값을 계산하는 문제다.
# 문제   
---------------------------------------

![Desktop View](/assets/img/Pointer14.png){: w="700" h="400" }

---------------------------------------

# 풀이

```c++
int main()
{
    int arr1[5];
    int arr2[5];
    int arr3[5];
    
    for (int i = 0; i < 5; i++)
        std::cin >> arr1[i];
    for (int i = 0; i < 5; i++)
        std::cin >> arr2[i];
    for (int i = 0; i < 5; i++)
        std::cin >> arr3[i];
    
    for (int i = 0; i < 5; i++)
    {
        std::cout << arr1[i] * arr2[i] + arr3[i] << " ";
    }
    
    return 0;
}
```
