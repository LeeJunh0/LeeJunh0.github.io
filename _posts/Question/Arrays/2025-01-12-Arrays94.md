# 풀이

```c++
#include <iostream>

void Check();

int arr[5];

int main()
{
    for (int i = 0; i < 5; i++)
        std::cin >> arr[i];
    
    Check();
    return 0;
}

void Check()
{
    int cur = 0;
    for (int i = 0; i < 5; i++)
    {
        if (cur < arr[i])
            cur = arr[i];
        else
        {
            std::cout << "증가안됨";
            return;
        }
    }
    
    std::cout << "증가됨";
}
```
---------------------------------------

# 후기

증가되는 순으로 숫자가 배열에 들어가 있는지 확인하는 문제다.

cur에 자신보다 큰 값이 들어올 때 값을 변경해 주고 작으면 return 해준다.
