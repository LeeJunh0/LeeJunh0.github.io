![Desktop View](/assets/img/Array10.png){: w="700" h="400" }

---------------------------------------

# 풀이

```c++
int YesOrNo();

int main()
{
    std::cout << YesOrNo();

    // 입력 : 5
    // 결과
    // 35

    return 0;
}

int YesOrNo()
{
    int input;
    std::cin >> input;

    if(input % 3 == 0)
        return 7;
    else if(input % 3 == 1)
        return 35;
    else
        return 50;
}
```
