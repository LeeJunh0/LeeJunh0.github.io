# 풀이

```c++
void Stairs();

char arr[11];

int main()
{
    std::cin >> arr;
    
    Stairs();
    return 0;
}

void Stairs()
{
    int idx = 1;
    int length = 0;
    for (int i = 0; arr[i] != '\0'; i++)
        length++;
    
    for (int i = 0; i < length; i++)
    {
        for (int j = 0; j < idx; j++)		
            std::cout << arr[length - j];
        
        std::cout << std::endl;
    }
}
```
---------------------------------------

# 후기

역순으로 출력양이 1개씩 늘어나는 느낌으로 계단을 만들면된다.

길이가 가장 끝을 가르킬태니 길이에서부터 시작해 증감식에 

idx를 최대개수로 정하고 이 값이 올라가면서 출력양이 증가한다.
