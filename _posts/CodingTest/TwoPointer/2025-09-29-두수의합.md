# 풀이

```c++
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cin.tie(NULL);
	cout.tie(NULL);

	int n;
	cin >> n;
	vector<int> vec(n);
	for (int i = 0; i < n; i++)
		cin >> vec[i];
	int x;
	cin >> x;

	int left = 0;
	int right = n - 1;
	int sum = 0;
	int count = 0;

	sort(vec.begin(), vec.end());
	while (left != right)
	{
		sum = vec[left] + vec[right];
		if (sum == x)
		{
			count++;
			left++;
		}
		else if (sum > x)
			right--;
		else
			left++;
	}

	cout << count << '\n';
	return 0;
}
```
---------------------------------------

# 후기

이번 문제는 투 포인터 문제다. 두 개의 포인터를 이용해 원하는 값을 만드는 것이다.

오름차순으로 정렬한 뒤 양쪽 끝에서부터 모든 경우의수를 체크해 풀었다.
