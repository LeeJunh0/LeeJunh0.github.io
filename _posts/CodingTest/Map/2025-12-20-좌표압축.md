# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cin.tie(NULL);
	cout.tie(NULL);

	int n;
	cin >> n;
	vector<int> inputArr;
	map<int, int> map;
	for (int i = 0; i < n; i++)
	{
		int input;
		cin >> input;

		map.insert({ input, 0 });
		inputArr.push_back(input);
	}

	int idx = 0;
	for (auto& cur : map)
		cur.second = idx++;

	for (int i = 0; i < n; i++)
		cout << map[inputArr[i]] << " ";

	return 0;
}
```
---------------------------------------

# 후기

맵은 항상 키값을 기준으로 순서를 지키는 레드블랙트리를 기반으로한다.

맵에 임의 인덱스 값 0을 넣고 중복을 중복을 받지않는 맵의 특성도 이용해 저장하고

입력값 순서를 저장해 인덱스값을 수정하고 입력값 순서에 따라 index를 출력했다.
