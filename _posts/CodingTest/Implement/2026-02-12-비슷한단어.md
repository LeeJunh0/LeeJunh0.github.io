# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cout.tie(NULL);
	cin.tie(NULL);

	int n;
	cin >> n;
	vector<string> vec;


	for (int i = 0; i < n; i++)
	{
		string str;
		cin >> str;

		string idxStr = "";
		vector<int> visited('z' - 'a' + 1, 0);
		int cur = 0;

		for (int j = 0; j < str.size(); j++)
		{
			if (visited[str[j] - 'a'] > 0)
				idxStr += visited[str[j] - 'a'];
			else
			{
				cur++;
				visited[str[j] - 'a'] = cur;
				idxStr += cur;
			}
		}

		vec.push_back(idxStr);
	}

	int result = 0;
	for (int i = 0; i < vec.size(); i++)
	{
		for (int j = i + 1; j < vec.size(); j++)
		{
			if (vec[i] == vec[j])
				result++;
		}
	}

	cout << result << '\n';
	return 0;
}
```

# 후기

패턴이 같은 문장의 쌍이 몇개인지 찾는 문제.

문장을 알파벳마다 나오는 순서를 카운팅하고 방문 체크를 한다.

이후 그 알파벳이 나오면 전에 순서를 카운팅했던 그 숫자를 문장에 넣어 순서문장으로 만들어서 풀었다.

abcd -> 1234, aacd -> 1123 이런식으로 말이다.
