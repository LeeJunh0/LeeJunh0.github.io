# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cout.tie(NULL);
	cin.tie(NULL);

	int n;
	cin >> n;
	int sum = 0;
	priority_queue<int> que;
	int dasom = 0;
	for (int i = 0; i < n; i++)
	{
		int num;
		cin >> num;
		sum += num;
		if (i == 0)
		{
			dasom = num;
			continue;
		}

		que.push(num);
	}

	int result = 0;
	while (que.empty() == false)
	{
		int cur = que.top();
		que.pop();

		if (cur < dasom)
			continue;

		cur--;
		result++;
		dasom++;
		que.push(cur);
	}

	std::cout << result << '\n';
	return 0;
}
```

# 후기

좀 더.. 정말 좀 더 정진하자 진짜 개발실력이 늘고싶다 정말 늘고싶다 늘고싶다 늘고싶다

이 문제는 다솜이를 제외한 후보들의 표를 최소한으로 깎아 다솜이가 1등이 되도록 만들면 되는 문제였다.

그래서 가볍게 다솜이랑 같거나 많은 애들을 한표씩 깎으며 다솜이에게 넣어주고

더이상 다솜이에 비비는 후보가 없을 때까지 도는 로직을 만들었지만 이 로직은 쓰레기였다.

왜냐하면 최소값을 구할수 없었다. 어떻게 구할수 없었냐. 이 문제는 "후보들 중 가장 많은 표를 가진 후보에게서 순회하며 한 표씩 가져와야 한다."

그렇지 않으면 한명에게서만 많은 표를 빼앗게 되는게 그건 최소값을 구하는 로직이 아니다.

그렇기에 우선순위 큐를 썼고 안쓰더라고 구현 할 수있는 문제다.
