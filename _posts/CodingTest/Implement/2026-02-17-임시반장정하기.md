# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>
#include <algorithm>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cout.tie(NULL);
	cin.tie(NULL);

	int n;
	cin >> n;
	vector<vector<int>> classVec(n + 1, vector<int>(6, 0));
	vector<vector<int>> cntVec(n + 1, vector<int>(n + 1, 0));;

	for (int i = 1; i <= n; i++)
	{
		for (int j = 1; j < 6; j++)
		{
			int classNumber;
			cin >> classNumber;
			classVec[i][j] = classNumber;
		}
	}
	
	for (int i = 1; i < 6; i++) 
	{
		for (int j = 1; j <= n; j++)
		{
			for (int k = 1; k <= n; k++)
			{
				if (j != k && classVec[j][i] == classVec[k][i])
					cntVec[j][k] = 1;
			}
		}
	}

	int result = 1;
	int maxCnt = 0;
	for (int i = 1; i <= n; i++)
	{
		int cur = 0;
		for (int j = 1; j <= n; j++)
			cur += cntVec[i][j];

		if (maxCnt < cur)
		{
			maxCnt = cur;
			result = i;
		}
	}

	cout << result << '\n';
	return 0;
}
```

# 후기

문제를 제대로 읽지 않았던 문제..

이 문제의 핵심은 다른 학생과 같은 반을 한 경험을 카운팅 하는것으로.

특정 학생이 다른 학생과 얼마나 많은 학생과 같은 경험을 한지가 중점인 문제가 아니다!

예를 들어 1번이 2, 3번과 1학년에 같은반을 한 경험이 있다면 그건 1, 2, 3번이 각각 1번씩을 카운팅 해야하지.

1, 2, 3번이 각각 2명의 학생과 경험한걸 카운팅하는것이 아니라는 뜻이다.. 이게 가장 중요하다.

이 부분만 깨닫게 된다면 금방 해결가능한 문제다. 그게 안되서 시간안에 못풀었다..
