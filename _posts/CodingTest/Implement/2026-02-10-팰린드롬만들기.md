# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cout.tie(NULL);
	cin.tie(NULL);

	string enName;
	cin >> enName;

	unordered_map<char, int> cntMap;

	sort(enName.begin(), enName.end());

	for (int i = 0;i < enName.size(); i++)
		cntMap[enName[i]]++;

	string front = "";
	string back = "";

	char centerCh = NULL;
	for (auto pair : cntMap)
	{
		if (pair.second % 2 == 1)
		{
			centerCh = pair.first;
			int idx = enName.find_first_of(centerCh);
			enName.erase(idx, 1);
			break;
		}
	}

	for (int i = 0; i < enName.size(); i++)
	{
		if (i % 2 == 0)
			front.push_back(enName[i]);
		else
			back.insert(back.begin(), enName[i]);
	}

	if (centerCh != NULL)
	{
		front.push_back(centerCh);
		enName.push_back(centerCh);
	}

	for (int i = 0; i < enName.size() / 2; i++)
	{
		if (front[i] == back[back.size() - i - 1])
			continue;

		cout << "I'm Sorry Hansoo" << '\n';
		return 0;
	}

	string result = front + back;
	cout << result << '\n';
	return 0;
}
```

# 후기

어우 구현실력이 점점 늘고있다..!!! 너무 재밌다.

이 문제는 앞에서 읽든 뒤에서 읽든 같아지는 문장을 만드는 문제다.

이 문제에 핵심은 홀수개의 문자를 찾아 관리하는게 핵심이라고 느꼈다.

이름의 길이가 홀수 일때, 앞뒤가 같은 문장은 홀수인 문자가 무조건 있다.

이름을 반으로 갈라 저장하고 검사한 후 중간에 있어야할 홀수 문자를 넣어주면 된다.
