# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>
#include <algorithm>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	int n;
	cin >> n;

	for (int i = 0; i < n; ++i)
	{
		string str;
		cin >> str;

		stack<char> stack;
		for (int j = 0; j < str.size(); ++j)
		{
			if (str[j] == '(')
				stack.push(str[j]);
			else if (stack.empty() == false)
			{
				if (stack.top() == '(' && str[j] == ')')
					stack.pop();
			}
			else
			{
				stack.push('.');
				break;
			}
		}

		if (stack.empty() == true)
			cout << "YES" << '\n';
		else
			cout << "NO" << '\n';
	}

	return 0;
}
```

# 후기

코테에서 스택문제로 유명한 괄호 문제다.

문제리스트를 보다가 생각해보니 괄호문제를 푼적이 있었나 싶었는데.. 역시 없었다.

그래서 이번 기회에 해결해보기로 했다.

괄호는 간단하게 생각해보면 왼쪽 오른쪽 개수를 세서 같으면 OK 라고 생각 할 수있는데

생각해보자 )))((( 도 왼쪽 오른쪽 개수는 같다 근데 올바른 괄호들이라 할 수 있나? 아니라는거다.

이제 문자열 처음부터 길이만큼 순회하는데

1. 처음에 오른쪽 괄호가 나오면 당연히 바로 NO를 출력하고 끝이다.
2. ( 일 경우 스택에 전부 넣어준다.
3. )일 경우 스택이 비어있는지 확인하고 비어있을경우 1번과 같은 경우다.
4. 스택에 ( 가 있는경우 pop을해 (를 제거하고 다음으로 넘어간다.

이 과정을 순회가 끝날때 까지 진행하고 스택이 비어있지 않으면 그 문자열은 NO고 비어있다면 YES를 출력한다.

마지막에 스택이 비어있는 확인 하는것으로 옳고 그름을 판단하기때문에

해결과정 3번에서 걸릴경우 스택에 ' . ' 을 채우는것으로 스택이 empty가 되지않도록 했다.
