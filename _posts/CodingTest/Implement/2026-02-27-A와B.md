# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>
#include <algorithm>

using namespace std;


int main()
{
	ios_base::sync_with_stdio(false);
	
	string a;
	string b;
	cin >> a;
	cin >> b;
	
	while (b.size() > 0)
	{
		if (b.size() == a.size())
		{
			if (a == b)
				cout << 1 << '\n';
			else
				cout << 0 << '\n';
			
			break;
		}

		char ch = b.back();
		b.pop_back();
		if (ch == 'B')
			reverse(b.begin(), b.end());
	}
	return 0;
}

```
---------------------------------------

# 후기

문자열 A를 정해진 규칙을 이용해 문자열 B를 만들수 있는지 알아내는 문제였다.

처음에는 무작정 고민에 고민을 하다 도저히 모르겠어서 DFS로 갈겨버렸다.

물론 테케야 바로 되겠지만, 시간초과가 바로났다 그럴거같았다. 하지만 그때는 줄일 방도가 떠오르지 않았다.

이 문제는 A -> B로 가는걸 생각하는것 과 B -> A로 되돌아 가는 걸로 접근하는게 훨씬 빠르고 간단한 문제였다.

생각해보면

규칙은 정해져있고 만들수 있는 문자열이 정해져있는 상태에서 처음 값도 있는 상태라는걸 생각해보면 된다.
