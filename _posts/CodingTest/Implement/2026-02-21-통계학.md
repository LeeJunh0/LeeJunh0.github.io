# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>
#include <algorithm>

using namespace std;

int main() 
{
	ios_base::sync_with_stdio(false);
	cin.tie(NULL);
	cout.tie(NULL);

	int n;
	cin >> n;
	vector<int> vec;
	unordered_map<int, int> cntMap;

	int sum = 0;
	int maxCnt = INT_MIN;
	for (int i = 0; i < n; i++)
	{
		int num;
		cin >> num;
		vec.push_back(num);
		cntMap[num]++;
		sum += num;
		maxCnt = max(maxCnt, cntMap[num]);
	}
	
	vector<int> result3;
	for (auto a : cntMap)
	{
		if (a.second == maxCnt)
			result3.push_back(a.first);
	}

	sort(result3.begin(), result3.end(), greater<int>());
	sort(vec.begin(), vec.end());

	double result1 = (double)sum / n;
	if (result1 < 0 && result1 > -0.5)
		result1 = 0;

	cout << round(result1) << '\n';
	cout << vec[n / 2] << '\n';
	if (result3.size() > 1)
		cout << result3[result3.size() - 2] << '\n';
	else
		cout << result3[0] << '\n';

	cout << vec[vec.size() - 1] - vec[0] << '\n';
	return 0;
}
```
---------------------------------------

# 후기

이 문제는 문제에서 제시한 조건의 값을 출력하는 문제였다.

1. 모든 입력값의 합 / n 값 (평균이 -0, -0.5 사이 일경우 -0이 나오는데 그 값은 0으로 치환)

2. 오름차순 정렬된 배열에서 중간에 해당하는 값

3. 입력값 중 가장 많이 등장하는 값 (여러 개 있는경우 그중 2번째로 작은값)

4. 최대값 - 최소값 

이 중 생각을 좀 했어야 했던 것이 1번이다. 사실 별거 없다.
