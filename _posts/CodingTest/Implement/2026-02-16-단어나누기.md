# 풀이

```c++
#include <bits/stdc++.h>
#include <unordered_map>
#include <algorithm>

using namespace std;

int main()
{
	ios_base::sync_with_stdio(false);
	cout.tie(NULL);
	cin.tie(NULL);

	string str;
	cin >> str;

	vector<string> vec;

	string left = "";
	string center = "";
	string right = "";

	for (int i = 0; i <= str.size() - 3; i++)
	{
		left += str[i];
		for (int j = i + 1; j <= str.size() - 2; j++)
		{
			center += str[j];
			for (int k = j + 1; k <= str.size() - 1; k++)
				right += str[k];

			string result = "";
			for (int k = 0; k < left.size(); k++)
			{
				if (left[left.size() - 1 - k] == '\0')
					continue;

				result += left[left.size() - 1 - k];
			}
			for (int k = 0; k < center.size(); k++)
			{
				if (center[center.size() - 1 - k] == '\0')
					continue;

				result += center[center.size() - 1 - k];
			}
			for (int k = 0; k < right.size(); k++)
			{
				if (right[right.size() - 1 - k] == '\0')
					continue;

				result += right[right.size() - 1 - k];
			}
			
			if (find(vec.begin(), vec.end(), result) == vec.end())
				vec.push_back(result);

			right = "";
		}
		center = "";
	}

	sort(vec.begin(), vec.end());
	cout << vec.front() << '\n';
	return 0;
}
```
---------------------------------------

# 후기

주어진 문장을 3분할 하는데 각 길이는 최소 1이상 이여야 한다.

3분할 한 단어들을 뒤집어 합치고, 이렇게 만들 수 있는 문장들중 사전 순으로 가장 빠른 문장을 출력한다.

핵심은 이거다.

문장의 3분할 경우의수를 전부 체크.

1. 3분할을 한다.
2. 그 단어들을 다 뒤집는다.
3. 그걸 합친다.
4. 중복인지 확인하고 저장한다.
5. 정렬하고 출력한다.

순으로 문제를 해결했다.
